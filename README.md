# Dropspot
<p align="center">
     <img align="center" width="400" src="/readme/dropspot_logo.svg">
</p>
A skate spot sharing app.

## Prerequisites

* GPS support

## Built With

* [Gradle](https://gradle.org//) - Dependency Management

## Features

USER
- Login 
- Register
- CRUD spots
- un/favorite spots
- vote for spot criteria
- navigate to spot
